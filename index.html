Особенности обмена

Администратор сайта может изменять информацию о товаре после обмена данными: изменять название товара, добавлять описание, фотографии, видео и сопутствующие товары. Цены товаров обновляются только посредством обмена данными с 1С. При редактировании цены товара на стороне сайта при следующей выгрузке из 1С эта цена затирается.

Добавление скидок для разделов и товаров производится на стороне сайта с помощью штатного функционала CMS 1C-Битрикс в разделе “Маркетинг – Товарный маркетинг – Правила работы с корзиной”.

При обмене данными сайта с 1С обновляются только цены и остатки, названия товаров, описания, фотографии и принадлежность к группе (разделу) не затрагиваются и не обновляются.

Из «1С» на сайт выгружаются:
  • Товары
  • Остатки с одного склада
  • Цены (один тип цены)
С сайта в «1С» выгружаются:
  • Новые заказы (номер заказа, ФИО, телефон, эл.почта)
  • Статус заказа (новый заказ)
  • Статус оплаты (ожидает оплату, оплачен)
